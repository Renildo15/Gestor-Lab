Gestor Lab - Sistema de Gerenciamento de Laboratórios

CERES - Campus Caicó


O Gestor Lab é um rojeto de Gestão de Laboratórios para gerenciar a equipe, as tarefas, os projetos, os artigos e as apresentações em no laboratório no LabCoffee MeetUp.

O Projeto foi desenvolvido para a disciplina de Engenharia de Software II. O sistema foi desenvolvido usando o framework Django e Python.

  • Visualiza Laboratório, Evento, horário, apresentação, TCC, estágio, projeto, artigo, pesquisa e etc.
  • Cadastra Laboratório, Evento, horário, apresentação, TCC, estágio, projeto, artigo, pesquisa e etc.
  • Deleta Laboratório, Evento, horário, apresentação, TCC, estágio, projeto, artigo, pesquisa e etc.
  • Edita Laboratório, Evento, horário, apresentação, TCC, estágio, projeto, artigo, pesquisa e etc.
  • Visualiza Laboratório, Evento, horário, apresentação, TCC, estágio, projeto, artigo, pesquisa e etc.
